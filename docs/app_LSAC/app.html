Introduction

DisApp-Tool is a WebApp that needs no Server Connection and runs in browser. DisApp-Tool has mapping capabilities and visualizes demo data in graphs.

DisApp-Tool uses
  • MorrisJS
  • OpenLayers for visualisation of data.
  • JavascriptClassCreator
for visualisation of data.
